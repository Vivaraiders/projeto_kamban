# Kanban CLI em Python

Projeto de gerenciamento de tarefas no terminal usando Python e MySQL.

## Funcionalidades

- Login de usuário
- Criar tarefas
- Deletar tarefas
- Atualizar status das tarefas
- Ordenação por status e prioridade

## Tecnologias

- Python
- MySQL
- SQL

## Como executar

1. Clone o repositório

git clone https://github.com/Vivaraiders/projeto_kamban/

2. Instale a dependência

pip install mysql-connector-python

3. Crie o banco usando

schema.sql

4. Execute o projeto

python main.py
